# Lianki

**Link + Anki = Lianki**

Lianki is a spaced repetition learning platform where you can review any URL using the scientifically-proven FSRS (Free Spaced Repetition Scheduler) algorithm. Perfect for reviewing YouTube videos, articles, documentation, or any web content you want to remember.

## What is Lianki?

Lianki combines the power of spaced repetition with the flexibility of the web. Instead of traditional flashcards, you can add any URL to your learning queue and review it at optimal intervals determined by the FSRS algorithm.

## Features

- 🔗 **Learn from URLs** - Add any web link (YouTube videos, articles, docs, etc.)
- 🧠 **FSRS Algorithm** - Scientifically optimized spaced repetition scheduling
- 📅 **Smart Reviews** - Review content at the perfect time for maximum retention
- 🎯 **Track Progress** - Monitor your learning journey and retention rates

## Tech Stack

- Next.js 15
- TypeScript
- ts-fsrs (FSRS algorithm implementation)
- Drizzle ORM
- NextAuth
- MongoDB/PostgreSQL

## Development

```bash
# Install dependencies
bun install

# Run development server
bun dev

# Build for production
bun build

# Start production server
bun start
```
