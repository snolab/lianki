# 초심자를 위한 일본어 학습 자료: 종합 가이드

일본어 학습을 시작하는 것은 많은 자료로 인해 압도적일 수 있습니다. 이 가이드는 절대 초심자부터 초기 중급 학습자까지의 최고의 자료를 학습 단계 및 자료 유형별로 안내합니다.

## 1단계: 문자 체계 (1-2주 차)

### 히라가나와 가타카나

문법이나 어휘에 뛰어들기 전에 두 개의 음성 문자 체계를 마스터해야 합니다.

**유튜브 채널:**
- [Learn Japanese with JapanesePod101.com - Hiragana Series](https://www.youtube.com/playlist?list=PLPSfPyOOcp3TNofiuTW5_S6l-qGrGHqUm)
  - 획순이 포함된 완전한 히라가나 레슨
  - 총 시청 시간 약 30분
- [Japanese Ammo with Misa - Hiragana](https://www.youtube.com/watch?v=6p9Il_j0zjc)
  - 명확한 발음 가이드
  - 기억법을 위한 연상 기호

**연습 도구:**
- [Tofugu's Hiragana Guide](https://www.tofugu.com/japanese/learn-hiragana/)
  - 연상 기호가 포함된 무료 종합 가이드
- [Real Kana](https://realkana.com/)
  - 대화형 연습 도구
- [Hiragana Quest](https://hiragana-quest.com/)
  - 게임화된 학습

**Pro Tip for Lianki:** 히라가나 비디오를 시청한 후, 검토 큐에 추가하세요. 각 문자를 매일 검토하여 유창하게 읽을 수 있을 때까지 연습하세요 (보통 1-2주의 연습이 필요).

## 2단계: 기본 문법 기초 (3-8주 차)

### JLPT N5 문법 (초급 레벨)

**유튜브 재생 목록:**
- [Japanese Ammo with Misa - Absolute Beginner](https://www.youtube.com/playlist?list=PLd5-Wp_4WsPU89IkN4v9lI_pEWBRyO_fu)
  - 40개 이상의 종합 문법 레슨
  - 예제를 통한 명확한 설명
  - 자기 학습자에게 적극 추천
- [Cure Dolly - Japanese From Scratch](https://www.youtube.com/playlist?list=PLg9uYxuZf8x_A-vcqqyOFZu06WlhnypWj)
  - 문법에 대한 독특한 유기적 접근
  - 일본어의 실제 작동 방식에 집중
- [Miku Real Japanese - JLPT N5 Grammar](https://www.youtube.com/playlist?list=PLCLBHbUvkRGo5AJwrulwhBmrit0-5TiXT)
  - 예: N5를 위한 100문장
  - 실용적인 문장 패턴

**교과서:**
- **Genki I** (워크북 포함)
  - 산업 표준 교과서
  - 대화가 포함된 체계적인 레슨
  - 체계적인 학습에 최적
- **Minna no Nihongo I**
  - 몰입 중심의 초점
  - 일본에서 인기
- **Tae Kim's Guide to Japanese**
  - [온라인 무료 가이드](http://www.guidetojapanese.org/learn/)
  - 문법 중심 접근
  - 훌륭한 참고 자료

**연습 웹사이트:**
- [Bunpro](https://bunpro.jp/)
  - 문법 포인트에 대한 SRS
  - Lianki 워크플로와 통합
- [Maggie Sensei](https://maggiesensei.com/)
  - 예제가 포함된 무료 문법 레슨
  - 실제 사용 설명

**Lianki 사용:** 각 문법 레슨 비디오를 완료할 때 Lianki에 추가하세요. 검토할 때 문법 포인트를 설명하고 예문을 구성할 수 있는지 확인하세요.

## 3단계: 필수 어휘 (진행 중)

### 핵심 어휘 목록

**JLPT N5 어휘 (~800단어):**
- [JLPT N5 Vocabulary List](https://jlptstudy.net/n5/?cat=1)
  - N5에 필요한 핵심 800단어
  - 카테고리별로 정리
- [Anki N5 Deck](https://ankiweb.net/shared/info/1679429599)
  - 사전 제작된 플래시카드 덱
  - Lianki에 URL을 가져올 수 있음

**빈도 목록:**
- [Core 2000 Japanese Words](https://iknow.jp/content/japanese)
  - 빈도별로 가장 많이 사용하는 단어
  - 자연스러운 학습 순서
- [WaniKani for Kanji](https://www.wanikani.com/)
  - 한자 및 어휘를 위한 SRS 시스템
  - 레벨 3까지 무료

**유튜브 어휘:**
- [JapanesePod101 - Must-Know Vocabulary](https://www.youtube.com/playlist?list=PLPSfPyOOcp3Qi8f9iG5M8qRvxjCTmP-Ed)
  - 필수 단어와 구문
  - 원어민 발음
- [Learn Japanese with Yuta - Daily Vocabulary](https://www.youtube.com/c/ThatJapaneseManYuta)
  - 실제 일본어
  - 문화적 문맥 포함

## 4단계: 청취 연습 (5주 차 이상)

### 초보자용 콘텐츠

**단계별 청취자:**
- [Comprehensible Japanese - Complete Beginner](https://www.youtube.com/c/ComprehensibleJapanese)
  - 느리고 명확한 일본어
  - 시각적 맥락
  - 초보자에게 완벽
- [Japanese Immersion with Asami](https://www.youtube.com/c/JapaneseImmersionwithAsami)
  - 자연스러운 발음, 명료한 음성
  - 자막이 있는 콘텐츠

**이야기 기반 학습:**
- [Nihongo con Teppei](https://nihongoconteppei.com/)
  - 초보자용 팟캐스트
  - 짧은 일일 에피소드
  - 무료 트랜스크립트 제공
- [Tadoku Graded Readers](https://tadoku.org/japanese/en/graded-readers/)
  - 클래스별 읽기 자료 무료 제공
  - 초보자용 레벨 0-5

**쉬운 뉴스:**
- [NHK News Web Easy](https://www3.nhk.or.jp/news/easy/)
  - 간단한 일본어 뉴스
  - 후리가나(읽기 보조)
  - 오디오 제공

## 5단계: 말하기 연습 (8주 차 이상)

**유튜브 대화 연습:**

- [HelloTalk](https://www.hellotalk.com/)

  - 무료 텍스트/음성 채팅
- [italki](https://www.italki.com/)
  - 전문 튜터
  - 저렴한 수업 (시간당 $5부터)

## Lianki와 함께 이 자료 사용하는 방법

### 단계별 워크플로:

1. **레슨 비디오 시청**
   - Lianki 유저스크립트 (Alt+F)를 사용하여 큐에 추가
   - 첫 번째 검토: 내일

2. **관련 자료 추가**
   - 연습 문제 추가
   - 예문 비디오 추가
   - 어휘 목록 추가

3. **검토 일정**
   - Lianki는 최적 간격을 위한 FSRS 알고리즘 사용
   - 예정일에 검토 (대시보드에서 확인)
   - 이해도 평가: 다시/어려움/좋음/쉬움

4. **점진적 확장**
   - 하루에 하나의 문법 포인트로 시작
   - 만나게 되는 어휘를 추가
   - 시간에 따라 자연스럽게 쌓아감

### 가져오기 전략:

**유튜브 재생 목록:**
YouTube API를 구성한 경우 [Learn 페이지](/learn)에서 전체 재생 목록을 한 번에 가져올 수 있습니다.

**사용자 지정 목록:**
URL(한 줄에 하나씩)을 포함한 텍스트 파일을 만들고 Learn 페이지에서 가져오기:
```
https://www.youtube.com/watch?v=VIDEO1
https://www.youtube.com/watch?v=VIDEO2
```

**수동 추가:**
YouTube를 탐색하고 나중에 검토할 비디오에 Alt+F를 사용하세요.

## 추천 학습 경로

### 1개월 차: 기초
- 1-2주 차: 히라가나 및 가타카나
- 3-4주 차: 기본 문법 (Genki Ch 1-3 또는 동등 수준)
- 매일: 10개의 새로운 어휘

### 2-3개월 차: 구축 블록
- 문법: Genki Ch 4-8 또는 동등 수준
- 어휘: 하루 10-15단어
- 청취: Comprehensible Japanese 시작
- 목표: JLPT N5 수준

### 4-6개월 차: 공고화
- 문법: Genki Ch 9-12 또는 동등 수준
- NHK Easy News 읽기 시작
- HelloTalk로 말하기 연습 시작
- 목표: N5에 능숙, N4 시작

## 성공을 위한 팁

1. **강도보다 일관성**
   - 매일 30분이 한 번에 3시간보다 낫습니다
   - Lianki를 사용하여 일관된 검토 유지

2. **기본을 놓치지 말 것**
   - 히라가나/가타카나를 마스터한 후 이동
   - 탄탄한 기초가 모든 것을 더 쉽게 만듭니다

3. **초기 몰입**
   - 이해하지 못하더라도 1주 차부터 듣기 시작
   - 뇌가 패턴을 인식하기 시작할 것입니다

4. **자신이 즐기는 자료 사용**
   - 애니메이션을 좋아하나요? 일본어 자막으로 시청하세요
   - 요리를 좋아하나요? 일본 요리 비디오를 시청하세요
   - 관심 = 동기

5. **Lianki로 정기적으로 검토**
   - FSRS 알고리즘은 최적의 기억 보장을 합니다
   - 시스템을 믿고 필요한 경우 검토하세요
