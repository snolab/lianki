# 将YouTube变成你的语言学校：Language Reactor + Lianki

如果你最喜欢的YouTube频道变成了你的私人语言导师呢？如果你看的每个视频都可以加速你的语言学习呢？通过合适的工具组合，这不仅可能——这是学习语言最有效的方法之一。

YouTube是语言学习者的宝藏：**以口语为导向的视频**（视频博客、采访、播客、教程）提供了比戏剧性的电影对话更清晰、更容易理解的语言。真实的人对着镜头讲话，解释概念，讲述故事——这是语言习得的最甜蜜的点。

今天，我想向你介绍**Language Reactor**，一个拥有超过200万用户的强大浏览器扩展，并向你展示如何通过**Lianki的间隔重复系统**来超强化它，从而形成一种让语言学习感觉不再像学习，而更像是观看你已经喜欢的内容的工作流程。

## 什么是Language Reactor？

[Language Reactor](https://www.languagereactor.com/)（前身为“Language Learning with Netflix”）是一个Chrome扩展，将视频流平台转变为互动语言学习环境。把它想象成语言学习者的瑞士军刀，想通过沉浸式学习来学习语言。

### 核心功能

**双语字幕**
杀手功能：Language Reactor可同时显示两种语言的字幕——你的母语和你正在学习的语言。你可以在不暂停视频的情况下即时连接不熟悉的单词和短语与其翻译。你甚至可以模糊目标语言字幕以先练习听力，然后悬停在任何单词上即可显示它。

**弹出式词典和词汇查询**
悬停在字幕中的任何单词上即可看到即时的定义、发音和例句。点击单词即可听它的发音，并获得指向外部词典的链接以进行更深入的探索。当前视频中的每个单词都会根据频率和学习阶段在侧面面板中实时更新排序。

**精确的视频控制**
Language Reactor添加了以下控制：
- 一键跳转字幕段落
- 自动循环困难句子
- 降低播放速度而不失真
- 保存特定时间戳以便稍后复习

**平台支持**
该扩展适用于：
- **YouTube**：数以千计的频道提供自动字幕支持——是可理解输入的最佳平台
- **Netflix及流媒体**：为影视节目提供完整的双字幕支持
- **网页内容**：导入带有机器翻译的文章和电子书
- **支持30多种语言**：从西班牙语和法语到日语、韩语、阿拉伯语等等

### 高级功能（€5.62/月）

免费版本提供核心的双字幕和词典功能。升级到Pro版可解锁：

**PhrasePump**：内置的间隔重复系统，从包含你保存词汇的句子中创建听力练习。它在引入新词汇的同时强化你标记为复习的单词。

**词汇导出**：将你的保存词汇列表连同上下文和定义导出为兼容Anki及其他SRS工具的格式。这是Lianki的用武之地。

**AI聊天机器人（Aria）**：一个AI语言伙伴，提供即时纠正，探索无尽话题，并帮助你建立会话信心。

**语音识别**：练习口语并获得发音的实时反馈。

## 为什么Language Reactor是个游戏改变者

传统语言学习应用通常在孤立中教授词汇——与任何有意义的背景无关的单词闪卡。Language Reactor颠覆了这一点：**你在真实内容中学习单词**，有情感背景、语气、面部表情和真实世界的使用模式。

研究持续表明，**可理解的输入**——大部分可以通过少量新单词理解的内容——是习得语言最有效的方法之一。Language Reactor让查找和消费可理解的输入变得轻而易举：

- 寻找有清晰对话的YouTube频道（视频博客、教程、采访）
- 从自然更易理解的面向口语的内容中学习，而不是脚本化的戏剧
- 观看理解度在70-80%的视频，通过双字幕学习其他部分
- 从真实媒体中构建词汇，而不是从课本例子中
- 通过自然使用的方式隐性学习语法模式

**为什么YouTube胜过电影**：YouTuber对着镜头说话，发音更清晰，自然地重复关键词，并一步一步解释概念。相比之下，一个在嘈杂餐厅场景中含糊的电影台词，博主每次都赢得理解。

根据语言习得研究，这种以沉浸式学习为基础的方法比传统学习方法能带来更好的记忆、更自然的发音和更快的整体进步。

## Lianki能带来什么？

Language Reactor解决了**输入**问题——它为你提供了理解的优秀内容和脚手架。但仅仅理解一次并不意味着你会记住。这就是**Lianki**的用武之地。

Lianki是一个现代间隔重复系统，基于[FSRS算法](https://github.com/open-spaced-repetition/fsrs4anki)构建——这是Anki 23.10+中使用的算法，被誉为比旧的SM-2系统更准确。FSRS使用一个复杂的记忆模型（难度、稳定性、可检索性）在最佳时刻安排复习——就在你快要遗忘之前。

### 为什么选择Lianki用于视频内容？

与传统的闪卡应用不同，Lianki是为**基于内容的学习**而设计的：

**基于URL的卡片**：用一个热键（`Alt+F`）添加任何网页——YouTube视频、文章、Netflix剧集、漫画阅读器——为一个闪卡。Lianki存储URL，当需要复习时，你会再次访问整个体验。

**以浏览器为先的工作流程**：安装[Lianki用户脚本](https://www.lianki.com/lianki.user.js)（Tampermonkey/Violentmonkey）并在当前页面进行复习。复习UI直接覆盖在你正在观看的内容上。

**键盘驱动的速度**：用`1-4`（重试/困难/良好/简单）或`HJKL`/`ASDT`评估你的记忆召回。用媒体键导航。无需鼠标，无需转换上下文。

**循环**：不是孤立地钻研词汇，而是循环整集、文章和视频。每次重温内容时，你理解更多。FSRS算法确保你在完美的间隔中复习——不太早（浪费时间），也不太晚（忘记所有）。

## 工作流程：Language Reactor + Lianki

以下是如何将这些工具结合成一个语言学习超级力量：

### 设置（5分钟）

1. **安装Language Reactor**
   从[Chrome Web Store](https://chromewebstore.google.com/detail/language-reactor/hoombieeljmmljlkjmnheibnpciblicm)获取。适用于Chrome、Edge及其他Chromium浏览器。

2. **安装Lianki用户脚本**
   访问[lianki.com/lianki.user.js](https://www.lianki.com/lianki.user.js)，通过Tampermonkey或Violentmonkey安装（用于运行用户脚本的免费浏览器扩展）。

3. **注册Lianki**
   访问[lianki.com](https://www.lianki.com)创建一个免费账户（通过电子邮件、GitHub或Google登录）。

### 核心循环

**步骤1：找到适合你水平的内容**

- 打开YouTube，在你的目标语言中搜索频道
- 查找以口语为导向的内容：视频博客、烹饪教程、采访、解释性视频
- 选择看起来有趣的内容——理想情况下，你已经理解70-80%的内容
- 专业提示：擅长解释的YouTubers（技术评论、操作视频、教育内容）使用的语言比娱乐节目更清晰

**步骤2：用Language Reactor观看**

- 开启双字幕（目标语言+你的母语）
- 正常观看视频，当遇到生词时使用弹出词典
- 将想要记住的单词保存到Language Reactor的词汇列表中
- 用Language Reactor的控件循环播放困难的句子直到弄明白

**步骤3：将视频添加到Lianki**

- 完成视频后，在YouTube页面上按`Alt+F`
- Lianki创建带有视频URL的卡片
- 评估你理解的程度：`1`（几乎没有），`2`（挣扎），`3`（良好），`4`（简单）
- Lianki使用FSRS安排你何时再次观看

**步骤4：以间隔重复复习**

- 几天或几周后（取决于你的评级），Lianki提醒你重看视频
- 打开你的Lianki复习队列在[lianki.com](https://www.lianki.com)
- 点击卡片→带回到YouTube视频
- 再次用Language Reactor观看（也许这次稍微隐藏字幕）
- 注意第二次观看时你理解了多少
- 再次评估你的回忆→Lianki调整下次复习的间隔

**步骤5：将词汇导出到Lianki（专业用户）**

如果你有Language Reactor Pro：
- 导出保存的词汇列表连同上下文
- 为关键词汇项创建个别的Lianki卡片
- 添加导出的句子作为URL卡片（如果它们链接到特定的时间戳）

### 为什么这有效：科学

**间隔重复+可理解输入=语言掌握**

此工作流程利用了认知科学中两个最有证明力的技巧：

1. **间隔重复**：FSRS算法在不断增加的间隔中安排复习，利用了间隔效应——发现信息在学习会话分散在一段时间内而不是集中在一起时更容易被记住。

2. **可理解的输入（i+1）**：语言学家斯蒂芬·克拉申的研究表明，当学习者接触到比当前水平略高的输入时，语言习得就会发生——通过一些努力是可以理解的。双字幕正好给你这个。

3. **丰富情境的学习**：认知心理学显示记忆是**依赖情境的**。当它们绑定于丰富的感官和情感背景时，你会记得更牢。一位YouTuber解释如何烹饪，一个视频博主谈论他们的一天——这些提供的背景远比单词闪卡丰富。

4. **测试效应**：每次你重温一个剧集并测试自己记住了什么时，你就在加强那些神经通道。Lianki的复习系统将这种提取练习融入你的工作流程中。

### 高级提示

**创建系列视频**

而不是添加个人视频，添加整个系列或播放列表：
- 从同一个YouTuber观看3-5个视频系列
- 批量添加所有URL到Lianki中（`Alt+Shift+V`进行批量添加）
- 当Lianki安排它们时，重看该系列以巩固特定主题的词汇

**多层次工具**

- 使用Language Reactor的PhrasePump进行每日词汇训练
- 使用Lianki进行内容循环（整个视频和频道）
- 两个系统互相强化：PhrasePump训练单词，Lianki在上下文中强化它们

**从双字幕过渡到单字幕**

首次观看：双字幕开启
第二次观看（几天后）：仅目标语言，点击查看母语翻译
第三次观看（几周后）：无字幕，测试你的纯听力理解

Lianki的FSRS调度自然在最佳间隔中将这些观看分开。

**追踪你的热图**

Lianki在你的主页上显示一个GitHub风格的活动热图。每次复习日成为一个绿色方块。用此来：
- 维持学习连击
- 可视化你的学习动力
- 发现模式（你在周末复习更多吗？）
