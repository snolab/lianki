# 初心者のための日本語学習教材：完全ガイド

_2026年2月25日投稿 [lianki.com](https://www.lianki.com)_

日本語を学び始めると、多くのリソースがあり圧倒されるかもしれません。このガイドでは、絶対初心者から中級レベルの早い段階の学習者向けに、学習ステージ別、リソースタイプ別に最適な教材を紹介します。

## ステージ1: 書記体系 (1～2週目)

### ひらがなとカタカナ

文法や語彙に取り組む前に、2つの音節文字を習得する必要があります。

**YouTubeチャンネル:**

- [Learn Japanese with JapanesePod101.com - ひらがなシリーズ](https://www.youtube.com/playlist?list=PLPSfPyOOcp3TNofiuTW5_S6l-qGrGHqUm)
  - 適切な筆順での完全なひらがなレッスン
  - 総視聴時間：約30分
- [Japanese Ammo with Misa - ひらがな](https://www.youtube.com/watch?v=6p9Il_j0zjc)
  - 明確な発音ガイド
  - 記憶補助のための記憶術

**練習ツール:**

- [Tofuguのひらがなガイド](https://www.tofugu.com/japanese/learn-hiragana/)
  - 無料の包括的ガイドと記憶術
- [Real Kana](https://realkana.com/)
  - インタラクティブな練習ツール
- [Kana Pro](https://kana.pro/)
  - モダンなインタラクティブかなトレーナー

**Liankiのプロヒント:** ひらがなのビデオを見た後、復習キューに追加します。各文字を流暢に読めるまで毎日復習します（通常1〜2週間の練習が必要です）。

## ステージ2: 基本文法基礎 (3～8週目)

### JLPT N5文法 (初心者レベル)

**YouTubeプレイリスト:**

- [Japanese Ammo with Misa - 絶対初心者](https://www.youtube.com/playlist?list=PLd5-Wp_4WsPU89IkN4v9lI_pEWBRyO_fu)
  - 40以上の総合的な文法レッスン
  - 例を交えた明確な説明
  - 自己学習者に非常に推奨
- [Cure Dolly - ゼロからの日本語](https://www.youtube.com/playlist?list=PLg9uYxuZf8x_A-vcqqyOFZu06WlhnypWj)
  - 文法へのユニークで有機的なアプローチ
  - 日本語の実際の働き方に焦点を当てる
- [Miku Real Japanese - JLPT N5文法](https://www.youtube.com/playlist?list=PLCLBHbUvkRGo5AJwrulwhBmrit0-5TiXT)
  - 例：N5のための100文
  - 実践的な文型

**教科書:**

- **げんき I**（ワークブック付き）
  - 業界標準の教科書
  - ダイアログ付きの構造化されたレッスン
  - 系統的な学習に最適
- **みんなの日本語 I**
  - より没入型のアプローチ
  - 日本で人気
- **Tae Kim's Guide to Japanese**
  - [無料のオンラインガイド](http://www.guidetojapanese.org/learn/)
  - 文法に焦点を当てたアプローチ
  - 優れた参考資料

**練習ウェブサイト:**

- [Bunpro](https://bunpro.jp/)
  - 文法ポイントのSRS
  - Liankiのワークフローと統合
- [Maggie Sensei](https://maggiesensei.com/)
  - 例を用いた無料の文法レッスン
  - 実生活での使用説明

**Liankiを使用して:** 各文法レッスンのビデオを完了する際にLiankiに追加します。復習時には文法ポイントを説明し、例文を構築できるようにします。

## Liankiを使用したこれらのリソースの使い方

### ステップバイステップワークフロー:

1. **レッスンビデオを見る**
   - Liankiユーザースクリプト（Alt+F）を使用してキューに追加
   - 初回レビュー：翌日

2. **関連教材を追加**
   - 練習問題を追加
   - 例文ビデオを追加
   - 語彙リストを追加

3. **レビュー予定**
   - Liankiは最適な間隔を保つためにFSRSアルゴリズムを使用
   - 期限が来たらレビュー（ダッシュボードを確認）
   - 理解度を評価：もう一度/難しい/良い/簡単

4. **段階的な拡大**
   - 1日1つの文法ポイントから始める
   - 語彙は出会った時に追加
   - 時間をかけて自然に増やす

### インポート戦略:

**YouTubeプレイリスト:**
YouTube APIが設定されている場合、[Learnページ](/learn)からプレイリスト全体を一度にインポートできます。

**カスタムリスト:**
URLが1行に1つずつあるテキストファイルを作成し、Learnページからインポートします：

```
https://www.youtube.com/watch?v=VIDEO1
https://www.youtube.com/watch?v=VIDEO2
```

**手動追加:**
YouTubeをブラウズし、後でレビューしたいビデオにAlt+Fを使用します。

## 推奨学習パス

### 月1: 基礎

- 1〜2週目: ひらがなとカタカナ
- 3〜4週目: 基本文法（げんき1〜3章または同等のもの）
- 毎日: 新しい語彙10語

### 月2〜3: 基礎構築

- 文法: げんき4〜8章または同等のもの
- 語彙: 毎日10〜15語
- リスニング: Comprehensible Japaneseから始める
- 目標: JLPT N5レベル

### 月4〜6: 強化

- 文法: げんき9〜12章または同等のもの
- NHK Easy Newsの読み始め
- HelloTalkとスピーキング練習を開始
- 目標: N5に慣れ、N4を始める

## 成功のためのヒント

1. **一貫性を重視**
   - 毎日30分が週1回3時間よりも効果的
   - Liankiを活用して一貫した復習を維持

2. **基礎を飛ばさない**
   - ひらがな/カタカナをマスターしてから進む
   - 確固たる基礎がすべてを容易にする

3. **早期没入**
   - 理解できなくても1週目からリスニングを始める
   - 脳がパターンを認識し始める

4. **楽しめる教材を使用**
   - アニメが好き？日本語字幕付きで見る
   - 料理が好き？日本の料理ビデオを見る
   - 興味 = モチベーション

5. **Liankiで定期的に復習**
   - FSRSアルゴリズムが最適な記憶を保証
   - システムを信頼し、指示されたときに復習

## 次は何をすべきか？

これらの初心者向け教材を完了した後（約3〜6か月）、以下に進む準備が整います：

- 中級リソース（N4/N3レベル）- _将来のブログ投稿で紹介予定_
- 字幕付きのネイティブコンテンツ
- チューターとの会話練習
- マンガやライトノベルの読書

## 今日から始めましょう

このリストから2〜3のリソースを選び、始めましょう。一度にすべてを使おうとせず、少数のリソースを一貫して使用する方が良いです。

**完全初心者向けクイックスタート:**

1. 今週ひらがなを学ぶ（Tofuguのガイド+Real Kanaを使用）
2. Misaの絶対初心者プレイリストを開始
3. ビデオをLiankiに追加し、毎日復習
4. JLPT N5リストから1日5〜10語の語彙を追加

覚えておいてください：日本語学習はマラソンであり、スプリントではありません。習ったことを忘れないようにLiankiを利用し、旅を楽しんでください！

**準備はできましたか？** [Learnページ](/learn)へ行って最初の教材をインポートするか、[Liankiユーザースクリプト](/lianki.user.js)をインストールして新たなビデオを見つけたらすぐに追加できるようにしましょう。
