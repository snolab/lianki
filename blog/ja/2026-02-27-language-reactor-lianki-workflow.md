# Netflixを言語の先生に変えよう：Language Reactor + Lianki

Netflixでの一気見セッションを構造化された言語学習の体験に変えることができたらどうでしょうか？ お気に入りのYouTubeチャンネルがあなた専用の言語家庭教師になったらどうでしょうか？ 適切なツールの組み合わせがあれば、これは可能なだけでなく、言語を学ぶ最も効果的な方法の一つです。

今日は、200万人以上が利用している強力なブラウザ拡張機能**Language Reactor**を紹介し、それを**Liankiの間隔反復システム**で強化して、学習が勉強ではなく、すでに好きなコンテンツを視聴するように感じられるワークフローを構築する方法をお見せします。

## Language Reactorとは？

[Language Reactor](https://www.languagereactor.com/)（旧称「Language Learning with Netflix」）は、ビデオストリーミングプラットフォームをインタラクティブな言語学習環境に変えるChrome拡張機能です。 言語学習者が没入型で学べるためのスイスアーミーナイフと考えてください。

### 主な機能

**二重言語字幕**
有力な機能：Language Reactorは、ネイティブ言語と学習中の言語のどちらの字幕も同時に表示します。ビデオを一時停止することなく、不慣れな単語やフレーズを翻訳と即座に結びつけることができます。さらに、まずリスニングを練習するためにターゲット言語の字幕をぼかし、任意の単語にホバーして表示することができます。

**ポップアップ辞書と単語検索**
字幕内の任意の単語にホバーすると、即座に定義、発音、例文が表示されます。単語をクリックして発音を聞くことができ、より深い探索のために外部辞書へのリンクを取得します。現在のビデオ内のすべての単語は、頻度と学習ステージによってリアルタイムに更新されるサイドパネルでランク付けされています。

**精密なビデオコントロール**
Language Reactorは次のコントロールを追加します：
- ワンクリックで字幕セグメント間をジャンプ
- 難しい文章を自動的にループ
- 音声を歪めることなく再生速度を遅くする
- 後でレビューするために特定のタイムスタンプを保存

**プラットフォームサポート**
この拡張機能は次のものと互換性があります：
- **Netflix**：完全な二重字幕サポート、難易度によるカタログ閲覧
- **YouTube**：自動字幕サポート付きの何千ものチャンネル
- **Webコンテンツ**：記事や電子書籍を機械翻訳でインポート
- **30以上の言語対応**：スペイン語やフランス語から日本語、韓国語、アラビア語など

### プロ機能（月額€5.62）

無料版では基本的な二重字幕と辞書機能が提供されます。プロ版にアップグレードすると以下が解放されます：

**PhrasePump**：保存した語彙を含む文章からリスニングエクササイズを作成する組み込みの間隔反復システムです。レビュー用にマークした単語を強化し、新しい語彙をコンテキストで紹介します。

**語彙エクスポート**：保存した単語リストをコンテキストと定義付きでAnkiやその他のSRSツールと互換性のあるフォーマットにエクスポートします。ここにLiankiが関わってきます。

**AIチャットボット（Aria）**：瞬時の修正を提供し、無限のトピックを探求し、会話の自信を築くのを助けるAI言語パートナー。

**音声認識**：発話を練習し、発音についてリアルタイムのフィードバックを受けます。

## なぜLanguage Reactorが革新的なのか

多くの伝統的な言語学習アプリは、語彙を孤立して教える傾向がありますが、Language Reactorはこれを逆にします：**あなたが興味を持つストーリーに埋め込まれている単語を学びます**。それは感情的なコンテキスト、トーン、表情、現実の使用パターンとともに学ぶことを可能にします。

研究は一貫して、ほとんど理解できる内容（理解可能なインプット）が最も効果的な言語習得方法の一つであることを示しています。Language Reactorは理解可能なインプットの発見と消費を容易にします：

- Netflixのカタログを言語と難易度でフィルタリング
- 70〜80％理解できるショーを視聴し、二重字幕で残りを学ぶ
- 本物のメディアから語彙を構築し、教科書の例ではない
- 自然に使われる文法パターンを暗黙的に学ぶ

言語習得研究によれば、このような没入型学習は、通常の学習方法と比較して、より良い定着率、より自然な発音、より速い全体的な進歩をもたらします。

## Liankiがもたらす価値とは？

Language Reactorは**インプット**問題を解決します。素晴らしいコンテンツにアクセスし、それを理解するための足場を提供します。しかし、一度理解したからといって必ずしも覚えているわけではありません。それが**Lianki**の役割です。

Liankiは、[FSRSアルゴリズム](https://github.com/open-spaced-repetition/fsrs4anki)に基づいた現代の間隔反復システムです。このアルゴリズムはAnki 23.10+で採用されており、以前のSM-2ベースシステムよりも大幅に正確であると称賛されています。FSRSは記憶（難易度、安定性、可逆性）の高度なモデルを使用して、ちょうど忘れかけたときにレビューをスケジュールします。

### なぜ動画コンテンツにLianki？

伝統的なカードゲームとは異なり、Liankiは**コンテンツベースの学習**用にデザインされています：

**URLベースのカード**：Netflixのエピソード、YouTubeビデオ、記事、マンガリーダーをワンキー（`Alt+F`）でフラッシュカードに追加できます。LiankiはURLを保存し、レビューの時に再度その体験に戻ります。

**ブラウザベースのワークフロー**：[Liankiユーザースクリプト](https://www.lianki.com/lianki.user.js)（Tampermonkey/Violentmonkey）をインストールして、現在のページを離れずにカードをレビューします。レビューUIは、視聴中のコンテンツの上に直接オーバーレイされます。

**キーボード駆動の速度**：`1-4`（再度/難しい/良い/簡単）や`HJKL`/`ASDT`でリコールを評価します。メディアキーでナビゲート。マウスは不要で、コンテキスト切り替えも必要ありません。

**ループ**：孤立した語彙をドリルする代わりに、エピソード、記事、ビデオ全体をループします。コンテンツを再訪するたびに、より多くを理解します。FSRSアルゴリズムは、ちょうど最適な間隔でのレビューを確保します（早すぎず、忘れる前に）。

## ワークフロー：Language Reactor + Lianki

これらのツールを組み合わせて、言語学習のスーパー・パワーに変える方法は次の通りです：

### セットアップ（5分）

1. **Language Reactorをインストールする**
   [Chromeウェブストア](https://chromewebstore.google.com/detail/language-reactor/hoombieeljmmljlkjmnheibnpciblicm)から入手します。Chrome、Edge、その他のChromiumブラウザで動作します。

2. **Liankiユーザースクリプトをインストールする**
   [lianki.com/lianki.user.js](https://www.lianki.com/lianki.user.js)にアクセスして、TampermonkeyまたはViolentmonkey（無料ブラウザ拡張機能）経由でインストールします。

3. **Liankiにサインアップする**
   [lianki.com](https://www.lianki.com)にアクセスして無料アカウントを作成します（メール、GitHub、またはGoogleサインイン）。

### 基本ループ

**ステップ1：自分のレベルのコンテンツを探す**

- Netflixを開き、Language Reactorの[カタログ](https://www.languagereactor.com/catalogue/netflix)を閲覧
- 言語と難易度（初級、中級、上級）でフィルタリング
- 既に70〜80％理解できる、興味のあるコンテンツを選ぶ

**ステップ2：Language Reactorで視聴**

- 二重字幕をオンにする（ターゲット言語＋ネイティブ言語）
- 新しい単語に出会ったときはポップアップ辞書を使いながら、通常通りエピソードを見る
- Language Reactorの語彙リストに覚えておきたい単語を保存
- Language Reactorのコントロールを使用して難しい文章をループして理解するまで繰り返す

**ステップ3：エピソードをLiankiに追加する**

- エピソード終了後、Netflixページで`Alt+F`キーを押す
- LiankiがエピソードのURLでカードを作成
- 理解度を評価する：`1`（ほとんど無理）、`2`（苦労した）、`3`（良い）、`4`（簡単）
- LiankiがFSRSを使用して再視聴タイミングをスケジュール

**ステップ4：間隔を空けてレビューする**

- 数日後または数週間後（評価に応じて）、Liankiがエピソードを再視聴するようにリマインド
- [lianki.com](https://www.lianki.com)でLiankiレビューキューを開く
- カードをクリック → 再びNetflixエピソードへ戻る
- Language Reactorで再視聴（今回は字幕を非表示にしてみる）
- 2回目の視聴でどれだけ理解が進んだかを実感
- 再度リコールを評価 → Liankiが次回のレビュー間隔を調整

**ステップ5：Liankiへ語彙をエクスポート（プロユーザー向け）**

Language Reactorプロを持っている場合：
- コンテキスト付きの保存済み語彙リストをエクスポート
- 重要な語彙項目一つ一つのLiankiカードを作成
- エクスポートされた文章をURLカードとして追加（特定のタイムスタンプにリンクしている場合）

### なぜこれが効果的か：科学的説明

**間隔反復 + 理解可能なインプット = 言語の習得**

このワークフローは認知科学における2つの最も実証されたテクニックを活用しています：

1. **間隔反復**：FSRSアルゴリズムは復習を増加する間隔でスケジュールし、情報が時間をかけて間隔をあけて学習セッションを行う方が、まとめて行うよりも良く保持されるというスペーシング効果を活用します。
