# YouTubeをあなたの語学学校に変えよう: Language Reactor + Lianki

お気に入りのYouTubeチャンネルがあなたの個人的な言語チューターになったらどうでしょうか？見るすべてのビデオが語学学習を加速させるとしたら？適切なツールの組み合わせを使えば、これは可能ですし、言語を学ぶ最も効果的な方法の1つです。

YouTubeは言語学習者にとっての宝の山です。**言語に特化したビデオ**（ビログ、インタビュー、ポッドキャスト、チュートリアル）は、ドラマチックな映画の台詞よりも明確で理解しやすい言葉を提供します。直接カメラに向かって話す人々、概念を説明し、物語を語る—これが言語獲得のための理想的な場所です。

今日は、強力なブラウザ拡張機能である**Language Reactor**を紹介し、**Liankiの間隔反復システム**と組み合わせて、言語学習を勉強ではなく、既に愛しているコンテンツを見るように感じさせるワークフローを紹介します。

## Language Reactorとは？

[Language Reactor](https://www.languagereactor.com/)（旧「Netflixで語学学習」）は、ビデオストリーミングプラットフォームをインタラクティブな語学学習環境に変えるChrome拡張機能です。言語学習者が没頭して学びたい場合に役立つスイスアーミーナイフのようなものです。

### メイン機能

**二重言語字幕**
この強力な機能により、Language Reactorはネイティブ言語と学習中の言語を同時に表示します。ビデオを一時停止することなく、不明な単語やフレーズを瞬時に翻訳と結びつけることができます。まず聞いて練習するために目的言語の字幕をぼかすこともでき、任意の単語にホバーすると表示されます。

**ポップアップ辞書＆単語検索**
字幕の任意の単語にホバーすると、即座に定義、発音、例文が表示されます。単語をクリックすると発音が聞こえ、さらに探求するための外部辞書へのリンクが提供されます。現在のビデオの中のすべての単語が頻度と学習ステージごとに横のパネルでリアルタイムに更新されます。

**精密ビデオコントロール**
Language Reactorは以下の操作を追加します：

- ワンクリックで字幕セグメント間をジャンプ
- 難しい文章を自動的にループ
- 音声を歪めずに再生速度を遅くする
- 後で見直すために特定のタイムスタンプを保存

**プラットフォームサポート**
拡張機能は以下で動作します：

- **YouTube**: 自動字幕サポートがある数千のチャンネル—理解しやすいインプットに最適なプラットフォーム
- **Netflix & ストリーミング**: ショーや映画の完全な二重字幕サポート
- **ウェブコンテンツ**: 話を機械翻訳でインポート
- **30以上の言語に対応**: スペイン語、フランス語から日本語、韓国語、アラビア語まで

### プロ機能 (€5.62/月)

無料版では基本的な二重字幕と辞書機能を提供します。Proにアップグレードすると以下がアンロックされます：

**PhrasePump**: 保存した語彙を含む文章からリスニングエクササイズを作成する間隔反復システム。レビュー用にマークした単語を強化し、新しい語彙を文脈で紹介します。

**語彙エクスポート**: 保存された単語リストをAnkiなどのSRSツールと互換性のある形式でエクスポートします。ここでLiankiが活躍します。

**AIチャットボット (Aria)**: 即座の修正を提供し、無限のトピックを探求し、対話力を高めるAI言語パートナー。

**音声認識**: 発音の練習をして、リアルタイムのフィードバックを受け取ります。

## Language Reactorがゲームチェンジャーである理由

従来の語学学習アプリは単語を孤立して教えることが多い—意味のある文脈から切り離されたフラッシュカードです。Language Reactorはこれを逆転させます：**実際のコンテンツに埋め込まれた単語を学びます**。感情的な文脈、トーン、表情、実際の使用パターンがあります。

研究により、**理解可能なインプット**—ほとんど理解でき、いくつか新しい単語が含まれるコンテンツ—が言語を習得する最も効果的な方法の1つであることが一貫して示されています。Language Reactorは、理解可能なインプットを見つけ、消費することを簡単にします：

- 明確で会話調のスピーチが特徴のYouTubeチャンネルを見つける（ビログ、チュートリアル、インタビュー）
- 自然に理解しやすい会話主体のコンテンツから学ぶ
- 70-80%理解できて、残りを二重字幕で学ぶビデオを視聴
- 教科書の例ではなく本物のメディアから語彙を構築
- 自然に使われる形で文法パターンを暗黙のうちに学ぶ

**なぜYouTubeが映画に勝るか**: ユーチューバーはカメラに向かって明確な発音で話し、自然にキーフレーズを繰り返し、概念を段階的に説明します。ビデオブロガーの「今日はパスタの作り方を教えます」が、騒がしいレストランシーンの中でつぶやかれる映画の台詞よりも理解しやすいのは明らかです。言語習得研究によると、こうした没入型学習は、従来の学習方法に比べてより良い定着、自然な発音、全体的な進歩の速さにつながります。

## Liankiが加えるものは何か？

Language Reactorは**インプット**の問題を解決します—素晴らしいコンテンツへのアクセスを提供し、それを理解するためのサポートがあります。しかし、一度理解することが覚えることを意味するわけではありません。ここで**Lianki**が登場します。

Liankiは、Anki 23.10+でも使用され、古いSM-2ベースのシステムよりもはるかに正確であると評価されているFSRSアルゴリズム（[FSRSアルゴリズム](https://github.com/open-spaced-repetition/fsrs4anki)）に基づいた最新の間隔反復システムです。FSRSは難易度、安定性、回想可能性の高度な記憶モデルを使用して、最適な瞬間—何かを忘れる直前にレビューをスケジュールします。

### なぜビデオコンテンツにLiankiを？

従来のフラッシュカードアプリとは異なり、Liankiは**コンテンツベースの学習**に設計されています：

**URLベースのカード**: 任意のウェブページ—YouTubeビデオ、記事、Netflixのエピソード、マンガリーダー—をフラッシュカードとして1つのホットキー（`Alt+F`）で追加します。LiankiはURLを保存し、レビューの時が来たら再びその体験を訪れます。

**ブラウザーファーストワークフロー**: [Liankiユーザースクリプト](https://www.lianki.com/lianki.user.js)（Tampermonkey/Violentmonkey）をインストールし、現在のページを離れることなくカードをレビューします。レビューUIはあなたが見ているものの上に直接表示されます。

**キーボード駆動の速度**: `1-4`（再試行/難しい/良い/簡単）または`HJKL`/`ASDT`でリコールを評価。メディアキーで移動。マウス不要、コンテキスト切り替えなし。

**ループ**: 孤立した語彙を drilledする代わりに、エピソード、記事、ビデオ全体をループします。コンテンツに再訪するたびに、より多くのことを理解します。FSRSアルゴリズムは完全な間隔でちょうど良い間隔でレビューをスケジュールします—早すぎません（時間を無駄にします）、遅すぎません（すべてを忘れます）。

## ワークフロー: Language Reactor + Lianki

以下はこれらのツールを組み合わせて語学学習のスーパーウェポンにする方法です：

### セットアップ（5分）

1. **Language Reactorをインストール**
   [Chrome Web ストア](https://chromewebstore.google.com/detail/language-reactor/hoombieeljmmljlkjmnheibnpciblicm)から入手。Chrome、Edge、その他のChromiumブラウザで動作します。

2. **Liankiユーザースクリプトをインストール**
   [lianki.com/lianki.user.js](https://www.lianki.com/lianki.user.js)にアクセスし、TampermonkeyまたはViolentmonkey（フリーのユーザースクリプト用ブラウザ拡張機能）を介してインストールします。

3. **Liankiにサインアップ**
   [lianki.com](https://www.lianki.com)にアクセスして、無料アカウントを作成（メール、GitHub、またはGoogleサインイン）。

### コアループ

**ステップ1: 自分のレベルに合ったコンテンツを見つける**

- YouTubeを開いてターゲット言語のチャンネルを検索
- 会話主体のコンテンツを探す：ビログ、料理チュートリアル、インタビュー、解説ビデオ
- 興味があるものを選ぶ—理想的にはすでに70-80%理解できるコンテンツ
- プロのヒント：解説するユーチューバー（技術レビュー、手順書、教育コンテンツ）はエンターテイメントよりも明確な話し方をします

**ステップ2: Language Reactorと一緒に視聴する**

- 二重字幕（学習中の言語＋ネイティブ言語）をオンにする
- ポップアップ辞書を使って新しい単語に遭遇したときにビデオを通常通り視聴
- 記憶したい単語をLanguage Reactorの語彙リストに保存する
- 難しい文章をLanguage Reactorのコントロールでループして感覚をつかむまで繰り返す

**ステップ3: ビデオをLiankiに追加**

- ビデオを見終わったら、YouTubeページで`Alt+F`を押す
- LiankiがビデオURLを含むカードを作成
- 理解度を評価：`1`（ほとんど）、`2`（苦労）、`3`（良い）、`4`（簡単）
- LiankiがFSRSを使って再視聴の時期をスケジュール

**ステップ4: 間隔を置いてレビュー**

- 評価に応じて数日または数週間後に、Liankiが再びビデオを観るように通知
- [lianki.com](https://www.lianki.com)でLiankiのレビューキューを開く
- カードをクリック→YouTubeビデオに戻る
- Language Reactorで再視聴（今回は字幕をもっと非表示にしてみる）
- 2回目の視聴で、どれだけ多く理解できるかに気付く
- 再び記憶を評価→Liankiが次のレビュー間隔を調整

**ステップ5: 語彙をLiankiにエクスポート（Proユーザー）**

もしLanguage Reactor Proがあれば：

- 保存した語彙リストを文脈と共にエクスポート
- キーとなる語彙アイテム用の個別のLiankiカードを作成
- エクスポートされた文をURLカードとして追加（特定のタイムスタンプにリンクしている場合）

### これが機能する理由：科学

**間隔反復 + 理解可能な入力 = 言語マスタリー**

このワークフローは認知科学における最も実証された技術を活用します：

1. **間隔反復**: FSRSアルゴリズムは、増加する間隔でレビューをスケジュールし、時期効果—情報が時間をかけて分散された学習セッションによってよりよく保持されるという発見を利用します。

2. **理解可能な入力（i+1）**: 言語学者Stephen Krashenの研究は、学習者が現在のレベルを少し上回る入力にさらされるときに言語習得が行われることを示しています—ほんの少しの努力で理解できるもの。二重字幕がこれを与えてくれます。

3. **文脈豊かな学習**: 認知心理学は記憶が**文脈依存**であることを示しています。豊かな感覚的・感情的文脈に結びつけられたときに物事をよりよく覚えます。料理の方法を説明するYouTuber、日々の出来事を話すビデオブロガー—これらは単語1つだけのフラッシュカードよりもはるかにリッチな文脈を提供します。

4. **テスト効果**: エピソードを再訪して何を覚えているかテストするたびに、これらの神経経路を強化しています。Liankiのレビュシステムはこの繰り返し練習をワークフローに織り込んでいます。

### 上級者向けのヒント

**ビデオシリーズを作成**

個々のビデオを追加する代わりに、全シリーズやプレイリストを追加：

- 同じユーチューバーの3-5のビデオシリーズを視聴
- すべてのURLをLiankiに一括追加する（`Alt+Shift+V`でバルク追加）
- Liankiがスケジュールしたとき、トピック固有の語彙を強化するためにシリーズを再視聴

**複数のツールをレイヤー**

- 毎日の語彙練習のためのLanguage ReactorのPhrasePumpを使用
- コンテンツループ（エピソード全体とチャンネル）のためのLiankiを使用
- 両システムが互いを強化：PhrasePumpで単語をドリル、Liankiで文脈で強化

**二重から単一字幕への進行**

1回目の視聴：両方の字幕オン
2回目の視聴（数日後）：ターゲット言語のみ、ネイティブ翻訳はクリックで表示
3回目の視聴（数週間後）：字幕なし、純粋なリスニング理解をテスト

LiankiのFSRSスケジューリングは、これらの視聴を最適な間隔で自然に配置します。

**ヒートマップを追跡**

LiankiはGitHubスタイルのアクティビティーヒートマップをホームページで表示します。各レビューの日が緑の四角になります。これを使って：

- 学習ストリークを維持
- 学びの勢いを視覚化
- パターンを見つける（週末にもっとレビューするか？）

**YouTubeチャンネルを追加**

Language ReactorはYouTubeで美しく動作します。チャンネル全体をカードとして追加：

- ターゲット言語を明確に話すYouTuberを見つける
- 最新のビデオURLをLiankiカードとして追加
- レビューするとき、そのチャンネルの新しいビデオを視聴
- 長期間にわたって、彼らのコンテンツを自然に理解できるようになります

### 実例：YouTubeでスペイン語を学ぶ

これがスペイン語を学ぶと仮定します：

**Week 1**: スペイン語の料理チャンネル（例えば「Cocina con Carmen」）を見つけます。Language Reactorを使ってパエリアの作り方のビデオを視聴。Liankiに追加し、`3`（良い）と評価。

**Week 2**: Liankiがリマインドします。パエリアビデオを再視聴—より多く理解できる、レシピのステップがより明確になりました。`4`（簡単）と評価。同じチャンネルから別の料理ビデオを追加し、`3`と評価。

**Week 3**: PhrasePumpが「sofreír」（炒める）、「fuego medio」（中火）、「añadir」（追加）についてドリル—すべて料理ビデオからの単語。それらを複数のビデオで何度も耳にします。

**Week 4**: Liankiが初めてのパエリアビデオをもう一度表示します。今回は字幕がほとんど必要ありません。`4`と評価。2番目のビデオが表示—今は簡単です。Carmenのチャンネル
